# 联结语句

## \*交叉联结

### **toys表**

| toy\_id | toy |
| :---: | :---: |
| 1 | hula hoop |
| 2 | balsa glider |
| 3 | toy soldiers |
| 4 | harmonica |
| 5 | baseball cards |

### boys表

| boy\_id | boy |
| :---: | :---: |
| 1 | Davey |
| 2 | Bobby |
| 3 | Beaver |
| 4 | Richie |

```
//交叉联结 ---> 组合
SELECT t.toy, b.boy
FROM toys AS t
    CROSS JOIN
    boys AS b;
```

## \*内联结与自然联结

| _**my\_contacts**_ | contact\_id\(主键\) | name | phone | email | prof\_id\(外键\) |
| :--- | :--- | :--- | :--- | :--- | :--- |
| _**profession**_ | prof\_id\(主键\) | profession |  |  |  |

```
//内联结语句，将profession表中的profession列根据prof_id联结进入my_contacts表
SELECT mc.name, mc.phone, p.profession
FROM my_contacts mc
    INNER JOIN
    profession p
ON mc.prod_id = p.prod_id;        /*根据这个条件来联结表,也可以根据条件，把=改成<>。*/


//自然联结，根据两张表中名字相同的列来内联结
SELECT mc.name, mc.phone, p.profession
FROM my_contacts mc
    NATURAL JOIN
    profession p;
```
